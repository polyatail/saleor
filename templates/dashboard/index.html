OrderWrite Company, User, and Product Creation

If products are being created for existing company, skip to the “Product Creation” step. If not, follow this guide from the top!

Company Creation:

Step 1: After logging with a “Staff” account, select the “Dashboard” menu in the top right of the page.

Step 2: Select the “Companies” tab on the left hand menu

Step 3: Select the blue circle with the “+”

Step 4: Enter the Company name in the “Name” field, and if needed the “Message to users (optional)” field. Any text in the “Message to users (optional)” field will be displayed above the catalog when customers log into their OrderWrite.

Step 5: Select “create” to enter the company into the database

User Creation:

Follow these instructions to create access for company employees and additional administrator accounts.

Step 1: Select the “User Accounts” tab in the left hand menu

Step 2: Create a username and password and select the appropriate company name from the drop down menu.

Step 3: If the user is going to be responsible for creating catalogue entries, select “Staff Status” and they will have access to the Dashboard. This should only be given to trusted users.

Step 4: If the user account is ready to launch, select the “Active” status. If not, select “Add” and the status can be updated at a later point.

Product Creation:

The first step to product creation is attribute creation. If you are unsure, start here.

Attribute Creation:

Step 1: Select the “Attributes” menu in the left hand menu

Step 2: If you see all of the variations or “attributes” needed for your product in the table, skip to “Product Types” step.

Step 3: Select the blue “+” in the top right corner.

Step 4: The “Display name” is what customers will see while shopping. Internal name can be used for storing more product specifics that may needed later.

Step 5: Select “Save” and repeat steps 1 through 4 until all attributes needed for the product are created.

Product Type Creation:

Creating product types allows you to group together different attributes to expedite the process next time!

Step 1: Select all attributes that will be used to create different variants of the same product under the “Attributes specific to each variant (optional)“, and select “Create”. That’s it!

Product Creation:

This is where you will create each individual variant of each product. Be sure to create each combination of variants available and label them with a unique SKU

Step 1: Select the “Products” tab in the left hand menu.

Step 2: Select the blue circle with the “+” in the top right corner.

Step 3: Select the appropriate “Product Type” from the prompted menu.

Step 4: Enter the Name for the product, and select the companies that you want it to be published to and select “Save”.

Step 5: In the next screen, select the “Add” button in the “Variant” table. Select a combination of each of your variants, and add a unique SKU. The “Variant name (optional)” field will be stored in the database, but not displayed to the customer. It can be used to store any extra information about the variant you need. Select the “Create” button and repeat step 5 until every combination of variants has been created.

Step 6: Selecting “Edit Images” at this page will allow you to drag and drop multiple images at once for upload.

Step 7: If the product is not ready for release, toggle the “Published” button in the top right corner.
